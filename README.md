## 📁 Installation

1. Clone the repository:

```bash
git clone https://github.com/MatheusIYoshida/Spotify-clone-menu.git
```

2. Navigate to the project directory:

```bash
cd Spotify-clone-menu
```

3. Open the projec:
- Open the `index.html` file directly in your browser
